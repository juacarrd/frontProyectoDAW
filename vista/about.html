Sobre esta aplicación

Esta plataforma ha sido desarrollada para facilitar la gestión y reserva de instalaciones deportivas del Ayuntamiento de Toro. Su objetivo es ofrecer a los ciudadanos un sistema moderno, accesible y eficiente para consultar la disponibilidad y reservar espacios deportivos municipales.

Tecnologías utilizadas

  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript (estructura MVC)
  • Backend: Node.js, Express y MongoDB
  • Autenticación JWT y pagos mediante PayPal Sandbox
